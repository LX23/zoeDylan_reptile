>#爬虫

>爬虫就是虫,抓a标签的

>爬虫设置里面,选择器需要定位到a标签上面去 支持简单的层级选择 (>、+、~、:**)这些符号别来 要报错(#hot a)

>**自动缓存数据：db/reptile.data.json**

>**/admin**是管理界面

>**可以配置邮件**

>邮件发送人在管理界面里面去修改

>邮件接收人在管理界面里面可以直接配置

>可以直接**node server/reptile**直接启用爬虫

>也可以**node index**启用网站(默认爬虫也会同时启用)

>**server/reptile.js**里面有配置文件说明

>**配置文件别乱改,不然等着报错吧**
